# CÁC THƯ VIỆN DÙNG CHO PROJECT

## 1. Thư viện chính cho MAX30102: SparkFun MAX3010x

Library ID: sparkfun/SparkFun MAX3010x Pulse and Proximity Sensor Library

Ưu điểm:

- Hỗ trợ cả MAX30102 và MAX30105

- Có sẵn các thuật toán cơ bản cho heart rate và SpO2

- Document tốt, ví dụ đầy đủ

- Được cộng đồng sử dụng rộng rãi

Cài đặt trong platformio.ini:

```ini
lib_deps = 
    sparkfun/SparkFun MAX3010x Pulse and Proximity Sensor Library
```

## 2. Thư viện OLED: Adafruit SSD1306

Library ID: adafruit/Adafruit SSD1306

Cài đặt:

```ini
lib_deps = 
    adafruit/Adafruit SSD1306
    adafruit/Adafruit GFX Library
```
